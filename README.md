# AI Agent Copilot — Retrieval-Augmented Multi-Agent System

## Overview
AI Agent Copilot is a production-style AI system that allows users to upload documents and ask questions using natural language.

It uses Retrieval-Augmented Generation (RAG) with a multi-step pipeline:
- Document ingestion
- Chunking
- Embeddings
- Vector search (ChromaDB)
- Answer generation

## Tech Stack
- Python
- FastAPI
- Streamlit
- LangChain
- ChromaDB
- Hugging Face

## Features
- Upload PDF documents
- Semantic search using embeddings
- Question-answering over documents
- Source citations
